### PHP Versie 3 oefenopdrachten en eindopdrachten


## Bugs

* Als je 2 keer dezelfde button drukt, haalt hij die weg en gaat er -2 van het aantalkaarten


## Hoe run je het
Je hebt een webserver nodig zoals bijvoorbeeld [XAMPP](https://www.apachefriends.org/download.html)
Je zet de files die je hiervan download in ```C:\XAMPP\htdocs\PHP 3```  Mocht PHP 3 nog niet bestaan, maak deze aan.
Je gaat naar localhost/php3 en nu kun je kiezen of je de oefenopdrachten of eindopdrachten wilt doen.
